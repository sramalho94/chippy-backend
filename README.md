### **_Getting Started_**

-   `Fork` and `clone`
-   `cd` into the directory
-   Run `npm i`
-   Run `sequelize db:create`
-   Run `sequelize db:migrate`
-   Run `sequelize db:seed:all`
-   Run `npm run dev`
